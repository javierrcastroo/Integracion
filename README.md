# Proyecto - Ingesta y Análisis de Vuelos en Streaming

- Tener Docker Desktop abierto
- En WSL hacer un directorio nuevo, y en ese directorio, escribir \texttt{code ..} en la terminal
- Una vez se abra VSCODE, en una terminal (ctrl+ñ), escribir git clone https://github.com/javierrcastroo/Integracion.git
- En esa misma terminal, hacer cd Integracion
- En ese directorio, ejecutar en la terminal ./run\_all.sh
- Dejar que todo se configure.
- Acceder a:

- texttt{http://localhost:8080} \hspace{0.2cm} (Kafka UI)
- texttt{http://localhost:8081} \hspace{0.2cm} (Mongo Express)
- texttt{http://localhost:8501} \hspace{0.2cm} (Dashboard en Streamlit)
- Ver como todo funciona de forma correcta.
- Para limpiar el entorno, ejecutar \texttt{reset.sh}.
